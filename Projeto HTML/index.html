Perguntas frequentes
FAQ
Quem fornece o laudo?
Médicos adequadamente resgistrados no conselho de classe, e altamente capacitados.
Quais equipamentos é preciso para solicitar o laudo?
É necessário um smartphone para baixar o aplicativo Raio X com Laudo e qualquer aparelho eletrocardiógrafo cujo exame possa ser enviado por uma única foto. Existe também a opção de solicitar os laudos através do computador, sendo necessário apenas se cadastrar e realizar o login.
É possível solicitar pelo computador?
Sim, é só se cadastrar e realizar o login no botão da barra superior dessa página.
Quem pode usar o App?
  • Médicos;
  • Clínicas e Policlínicas assistenciais;
  • Hospitais privados;
  • Sistemas públicos de saúde (Hospitais, Municípios, Estados, etc);
  • Setores de medicina do trabalho que requerem avaliações admissionais e demissionais.
É necessário que a solicitação seja feita por um médico?
Sim, por motivos de segurança, é solicitado o CRM, o registro no Conselho Regional de Medicina, para realizar o cadastro tanto pelo site quanto pelo aplicativo.
Há algum tempo obrigatório de uso?
Não. Após concordar com os termos, você usa o aplicativo enquanto tiver demanda, não existe utilização mínima e nem multa caso deixe de usar o Raio X com Laudo.
Como é feito o pagamento?
O pagamento é feito no ato da solicitação do laudo, através de PIX ou TED.
Quanto custa?
Cada laudo custa R$ 15,00 reais. Mas o primeito é grátis, faça o teste!
Preciso imprimir o laudo para entregar ao paciente?
O laudo é recebido pelo e-mail ou aplicativo, com isso, é possivel enviar o laudo digitalmente para o paciente excluindo a necessiadade de impressão. A natureza agradece!
